# GOV-NO-STRATEGY-DELEGATION

Status: Active
Authority: Strategic Authority
Implementation: Strategic Guardian

Rule:

Strategic Guardian owns:

- classification
- path selection
- corridor selection
- budget validation
- success condition definition

Execution Chat owns:

- execution
- evidence retrieval
- evidence reporting
- bounded verification

Execution Chat must not be instructed to:

- determine validation strategy
- choose validation paths
- compare approaches
- identify lowest-cost methods
- decide investigation scope

Operational Principle:

Guardian Decides.
Execution Executes.
