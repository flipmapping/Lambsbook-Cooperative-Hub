# Authority Namespace Rule v1.0

Status

ACTIVE

Effective

Immediately

Repository

Growth Engine

--------------------------------------------------

Purpose

Define the canonical authority namespace used by all execution packages.

This document is repository-backed Repository Truth.

--------------------------------------------------

Grandfather Rule

The following authority families remain immutable historical records:

- FDR-010Exx
- RMP-010Exx

Execution SHALL preserve their lineage, certifications and traceability.

Execution SHALL NOT:

- renumber;
- migrate;
- reinterpret;
- merge

legacy authorities.

--------------------------------------------------

Namespace Rule

All NEW implementation authorities SHALL use stream-prefixed namespaces.

Growth Engine

GE-RMP-###

Main Cooperative Hub

HUB-RMP-###

Open Brain / EOS

EOS-RMP-###

Infrastructure

INF-###

--------------------------------------------------

Independent Numbering Rule

Each authority stream owns an independent sequence.

Examples

GE-RMP-001
GE-RMP-002
GE-RMP-003

HUB-RMP-001

EOS-RMP-001

INF-001

Numbers SHALL NOT be shared between streams.

--------------------------------------------------

Work Package Identity Rule

A work package inherits one implementation authority.

That authority SHALL remain identical throughout:

Founder Decision

Execution Derivation

Infrastructure Authority

Implementation Authority

Claude Instruction Brief

Claude Package

Python Repository Mutation Package

Founder Execution Package

Certification

Release Synchronization

--------------------------------------------------

Authority Consistency Rule

Exactly one implementation authority SHALL govern one work package.

Mixed authorities constitute a BLOCKED condition.

--------------------------------------------------

Collision Prevention Rule

Parallel execution chats SHALL allocate authorities only within their own stream.

Examples

Growth Engine
    GE-RMP-###

Main Cooperative Hub
    HUB-RMP-###

Open Brain
    EOS-RMP-###

Infrastructure
    INF-###

Parallel streams SHALL NOT coordinate numbering.
