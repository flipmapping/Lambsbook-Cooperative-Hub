# CMA-002B

Status

ACTIVE

Title

Content Materialization Report (CMR)

Purpose

Record deterministic provenance for every runtime content artifact
generated by the Content Materialization Authority.

Required Report Entries

For every generated JSON file record:

• Source repository artifact(s)
• Extracted factual content
• Destination JSON file
• Translation locale
• Unresolved items
• Validation status

Success Criteria

Every runtime JSON file is traceable back to one or more
Founder-approved repository artifacts.

No generated content may lack provenance.

Builder SHALL consume only repository-certified runtime content
approved through this report.
