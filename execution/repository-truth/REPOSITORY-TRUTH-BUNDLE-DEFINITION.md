# Repository Truth Bundle Definition

Status

CANONICAL

Repository

Lambsbook-Open-Brain

Authority Stream

EOS

Type

Execution Infrastructure

--------------------------------------------------

Definition

A Repository Truth Bundle (RTB) is the certified execution context
derived from certified Repository Truth.

The RTB defines the bounded repository surface required to perform
a specific repository mutation.

--------------------------------------------------

Repository Truth

Repository Truth establishes objective facts about the repository.

--------------------------------------------------

Execution Context

Execution Context establishes the minimum repository evidence
required to perform a bounded implementation.

--------------------------------------------------

Certification

An RTB SHALL be derived only from certified Repository Truth.

--------------------------------------------------

Builder Relationship

Builder v1.0 SHALL consume the RTB.

Builder v1.0 SHALL NOT create the RTB.

--------------------------------------------------

Claude Relationship

Claude SHALL use the RTB only to perform the bounded repository
mutation authorized by the associated Implementation Authority.

--------------------------------------------------

Expected Outcome

A deterministic and certifiable execution context suitable for
bounded repository mutation.
