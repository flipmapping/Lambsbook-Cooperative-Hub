# EOS-GOV-001

Status

ACTIVE

Effective

Immediately

Purpose

Define constitutional governance for the Execution Operating System after
Builder v1.2 Operational Certification.

Governed Constitutional Components

• Execution Operating System
• Builder v1.2
• Builder Invocation Standard (BIS)
• Implementation Authority Standard (IAS)
• Content Materialization Authority (CMA)

Governance Principles

Execution infrastructure SHALL evolve only through explicit Founder-approved
constitutional authorities.

Ordinary implementation authorities SHALL consume the Execution Operating
System but SHALL NOT redefine it.

Change Classification

Constitutional Changes

• Execution Operating System
• Builder
• BIS
• IAS
• CMA

Implementation Changes

• Product capabilities
• Runtime behaviour
• UI
• APIs
• Business workflows

Rule

Implementation authorities SHALL reference constitutional artifacts rather
than copying or redefining them.

Relationship

Execution infrastructure provides the stable operating environment for
implementation authorities.

Implementation authorities deliver product value within that environment.
