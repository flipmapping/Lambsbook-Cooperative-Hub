# BLD-RES-001

Status

APPROVED

Title

Package Resource Manifest (PRM)

Classification

Builder Enhancement

Purpose

Introduce a generic Package Resource Manifest capability that allows Builder
to package immutable repository-certified resources explicitly declared by
the implementation authority.

Constitutional Principle

Builder SHALL package only repository-certified resources explicitly declared
by the implementation authority.

Builder SHALL NOT discover, interpret, transform, summarize, generate,
or derive packaged resources.

Builder SHALL preserve all existing Package Integrity behaviour.

This enhancement SHALL NOT modify Builder v1.2 Operational Qualification.

Target Release

Builder v1.3
