# Claude Instruction Brief (CIB) Templates

Status: APPROVED

## Purpose

This directory contains the canonical Claude Instruction Brief templates used to generate Repository Mutation Packages.

## Canonical Templates

### Production Authorities

- CIB-PROD-v1.0.md

Used for:

- new Production Surfaces
- new Workspaces
- new Runtime capabilities

---

### Corrective Authorities

- CIB-CORR-v1.0.md

Used for:

- corrective refinements
- bounded implementation corrections
- preservation-oriented Repository Mutation Packages

---

## Template Governance

These templates are the canonical repository baseline.

Future CIB revisions SHALL supersede these templates by semantic version.

Previous versions SHALL remain preserved for historical traceability.
