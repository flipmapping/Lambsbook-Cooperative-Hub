Title

Applicant Activity Center Gap

Repository Truth

The repository already provides admissions activity runtime capabilities, including applicant activity listing.

Internal admissions workspaces already consume this runtime.

No applicant-facing activity center currently exists.

Applicants cannot view the chronological activity history of their admissions process from the public Growth Engine experience.

Conclusion

The next bounded Repository Reality mutation is an Applicant Activity Center that consumes the existing admissions activity runtime without backend or database mutation.
