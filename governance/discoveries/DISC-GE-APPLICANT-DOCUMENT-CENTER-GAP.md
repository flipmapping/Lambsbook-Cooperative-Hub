Title

Applicant Document Center Gap

Repository Truth

The repository already provides admissions document runtime capabilities, including document creation, listing, update, and archive operations.

No applicant-facing document center currently exists.

Applicants cannot view required documents, submitted documents, or upload status from the public Growth Engine experience.

Conclusion

The next bounded Repository Reality mutation is an Applicant Document Center that consumes the existing admissions runtime without backend or database mutation.
