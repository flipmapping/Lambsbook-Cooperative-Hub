Title

Applicant Lifecycle Timeline Gap

Repository Truth

The repository already provides admissions lifecycle event runtime capabilities, including lifecycle event listing for each prospect.

Internal admissions workspaces already consume this runtime.

No applicant-facing lifecycle timeline currently exists.

Applicants cannot view the chronological history of their admissions journey from the public Growth Engine experience.

Conclusion

The next bounded Repository Reality mutation is an Applicant Lifecycle Timeline that consumes the existing admissions lifecycle event runtime without backend or database mutation.
