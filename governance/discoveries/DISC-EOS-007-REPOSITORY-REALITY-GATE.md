Repository Reality Gate (RRG)

Repository Truth

GE-RMP-005 demonstrated that Builder successfully produced a Claude Package while Repository Reality remained unchanged.

Discovery

A new Implementation Authority SHALL correspond to exactly one new bounded repository mutation.

If Repository Truth already satisfies all authorized deliverables:

- no successor authority SHALL be derived;
- Builder SHALL NOT be invoked merely to produce a differently named package;
- the previous authority remains the latest Repository Reality change.

Conclusion

Repository Reality is the governing precondition for Implementation Authority derivation.
