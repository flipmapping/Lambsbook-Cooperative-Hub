Title

Applicant Follow-up Task Center Gap

Repository Truth

The repository already provides admissions follow-up task runtime capabilities, including applicant follow-up task listing.

Internal admissions workspaces already consume this runtime.

No applicant-facing follow-up task center currently exists.

Applicants cannot view assigned admissions follow-up tasks from the public Growth Engine experience.

Conclusion

The next bounded Repository Reality mutation is an Applicant Follow-up Task Center that consumes the existing admissions follow-up task runtime without backend or database mutation.
